Box Model

Each section uses margin, padding, and borders to create efficient spacing and layout.

The Box Model is composed of 4 components:

  • Content

    This is the innermost area where the actual content of the element resides.

  • Padding

    This is a transparent area surrounding the content. It creates space between the content and the element's border.

  • Border

    This is a visible line that surrounds the padding and content.

  • Margin

    This is a transparent area outside the border. It creates space between the element's border and other neighboring elements on the page
